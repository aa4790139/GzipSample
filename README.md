# GzipSample
C# 与 JS  之间Gzip互通Demo

#### 目录结构
GzipSample
- csharp：源码工程
- js：源码工程

#### 包含具体内容
- JS 压缩 Gzip demo
- JS 压缩 Gzip + base64 demo
- JS 解压 CSharp 的 Gzip demo
- JS 解压 CSharp 的 base64 + gzip
- CSharp 压缩 Gzip demo
- CSharp 压缩 Gzip + base64 demo
- CSharp 解压 JS 的 Gzip demo
- CSharp 解压 JS 的 base64 + gzip
- Base64 encode 和 decode
